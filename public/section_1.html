Getting Started

This initial section will cover the basic tools needed and how to create and run a “Hello World” NodeJS Project.

Tools

NodeJS

Node.js is an open-source, cross-platform, back-end JavaScript runtime environment that runs on the V8 engine and executes JavaScript code outside a web browser. Node.js lets developers use JavaScript to write command line tools and for server-side scripting.

Installing Node.js depends on the operating system you are using. Taking a look at https://nodejs.org/en/download/ is a good place to start. Here we will document the installation process through a Linux (Ubuntu) terminal.

  1. Open the terminal
  2. Install nodejs with the following command: apt install nodejs
  3. Check to see if the installation was successful: node -v

Npm

Npm is a package manager for the JavaScript language and the default package manager used by Node.js. If you installed Node.js using an installer then npm has already been added to your computer. Otherwise you can install it separately like this:

  1. Open the terminal
  2. Install npm with the following command: sudo apt install npm
  3. Check to see if the installation was successful: npm -v

A text editor

This will be the editor used to write the code in. Anything works, really but a text editor specially made for code is recommended and my personal choice is Visual Studio Code: https://code.visualstudio.com/

A command line interface

A command-line interface processes commands to a computer program in the form of lines of text. No matter what operating system you are on, you have access to a CLI. On Windows you can use either Command Prompt or Powershell. On Linux and Mac you have the Terminal.

Hello World!

To create the most basic NodeJS project possible is as easy as following these steps:

  1. Create a file called hello.js
  2. In the file write the following code which will print “Hello World” to the console: console.log(“Hello World”);
  3. Open a terminal and navigate to the directory where your JavaScript file is located
  4. To run the file with NodeJS use the command: node hello.js
